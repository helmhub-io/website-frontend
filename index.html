How to Get Started

1

Install Helm

Download and install Helm CLI on your local machine or CI/CD environment.

curl https://get.helm.sh/helm-v3.13.0-linux-amd64.tar.gz | tar -xz
2

Browse Charts

Explore our comprehensive collection of Helm charts to find the right application for your needs.

Browse All Charts
3

Deploy Charts

Search, customize, and deploy charts to your Kubernetes cluster with helm install.

helm install my-release helmhub/nginx
